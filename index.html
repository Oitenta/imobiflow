Configurações Google Cloud

Ative seu ecossistema seguro no Google Drive

Diagnóstico: Execução Local Detectada (file://)

Como você está abrindo o arquivo diretamente do seu computador, o login da Google Cloud **não funcionará** devido às regras de segurança de redirecionamento do Google.

Como fazer funcionar:

  • Publique o arquivo no **GitHub Pages** de forma gratuita.
  • Rode um servidor local básico no seu editor de código (ex: extensão **Live Server** no VS Code).
  • Isso garantirá um endereço oficial (ex: http://localhost:5500) que a API da Google aceita para logins.

Por padrão, o app roda usando chaves demonstrativas públicas de sandbox. Para sua segurança e privacidade permanentes, configure suas próprias chaves do Google Cloud:

Como criar minhas chaves gratuitas?

  1. Acesse o Google Cloud Console.
  2. Crie um projeto rápido chamado "ImobiFlow".
  3. Vá em APIs e Serviços > Ative a API do Google Drive API.
  4. Na aba Tela de consentimento OAuth, configure como de tipo "Externo".
  5. Em Credenciais, crie uma Chave de API e um ID do cliente OAuth.

Integração com Google Sheets

Trabalhe e analise seus Leads direto em planilhas virtuais

Alternativa Offline Instantânea

Gere um arquivo que você pode importar no Excel ou Sheets

Para fazer as informações do App irem diretamente para a planilha, siga os seguintes passos:

  1. Crie uma nova Planilha Google.
  2. No menu superior, vá em Extensões > Apps Script.
  3. Substitua todo o código que aparecer lá pelo código fornecido abaixo.
  4. Clique no botão de salvar e depois em Implantar > Nova implantação.
  5. Escolha o tipo: App da Web. Em "Quem pode acessar", mude para "Qualquer pessoa".
  6. Copie o link do App da Web fornecido pela implantação e cole no campo "URL da API" acima.
Código Apps Script do Google: 
/* Código para o Apps Script do Google Sheets */
function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    
    sheet.clearContents();
    sheet.appendRow(["ID", "Data Criacao", "Lead", "WhatsApp", "Corretor", "Imovel", "Valor (R$)", "Fase", "Temperatura", "Acompanhamento", "Tags", "Observacoes"]);
    
    data.leads.forEach(function(lead) {
      sheet.appendRow([
        lead.id,
        lead.date || "",
        lead.name || "",
        lead.whatsapp || "",
        lead.agent || "",
        lead.property || "",
        lead.value || 0,
        lead.stage || "",
        lead.priority || "",
        lead.followUpDate || "",
        (lead.tags || []).join(", "),
        lead.notes || ""
      ]);
    });
    
    return ContentService.createTextOutput(JSON.stringify({status: "success", count: data.leads.length}))
      .setMimeType(ContentService.MimeType.JSON);
  } catch (error) {
    return ContentService.createTextOutput(JSON.stringify({status: "error", message: error.toString()}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

function doGet() {
  return ContentService.createTextOutput("A API do ImobiFlow está pronta!")
    .setMimeType(ContentService.MimeType.TEXT);
}
